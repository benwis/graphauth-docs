Upload file to S3 storage and create a DB resource to track it in the resources DB. Must be logged in or
a platform admin, and the feature must be enabled for your tenant.

```graphql
uploadResource(
  tenantName: String!
  id: UUID!
  resourceName: String!
  file: Upload!
): Boolean!
```
