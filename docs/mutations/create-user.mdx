Creates user based on username, email, and an optional password. Stores that user in the DB.
If a username is not provided, an alternate login method will be required, whether that's
magic link or webauthn.

```graphql
createUser(
  tenantName: String!
  name: String!
  displayName: String
  password: String
  email: String!
  role: String!
): User!
```
