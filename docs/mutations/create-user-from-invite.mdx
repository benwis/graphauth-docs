Creates user that has an invite token with a variety of other optional user details.
Probably not recommended unless you want to register an address, phone, or other details.

```graphql
createUserFromInvite(
  inviteToken: String!
  name: String!
  displayName: String
  password: String
  role: String!
  userDetails: Boolean!
  phone: String
  company: String
  brand: String
  vcService: String
  vcUsername: String
  addressDetails: Boolean!
  nameLine: String
  line1: String
  line2: String
  line3: String
  city: String
  state: String
  country: String
  postalCode: String
): User!
```
