Gets Queries the system for a particular user. Can be searched for with user id, user name,
or user email. Will only return results from the tenant specified, and you must be a member
of that tenant with sufficient permissions to use this query.

```graphql
user(
  tenantName: String!
  id: UUID
  name: String
  email: String
): User
```
